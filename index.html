Telecom Industry Customer Churn Prediction Model

I developed a simple customer churn prediction model for my Machine Learning module. Since this was a classification type of problem which will be used to predict if the customer will churn or stay, I used Logistic Regression which is a supervised learning model.

High-level overview of what was done:

  • Found a dataset on Kaggle
  • Did data cleaning by removing duplicate rows, removing anomalies, doing feature engineering
  • Did one-hot encoding and label encoding as required
  • Did univariate and multivariate analysis to understand the dataset
  • Since this project was restricted to using the scikit learn library for the models, I used Decision Tree Classifier, Logistic Regression, Random Forest Classifier, K Neighbors Classifier, Gaussian NB(Naive Bayes) and Support Vector Classification (SVC) algorithm as my base models from which I did standard feature scaling
  • Split the data into traiing and testing sets
  • Did data augmentation (synthetic oversampling) with SMOTE as the data was imbalanced
  • Did grid search to further tune the hyperparameters of the model
  • Retrained the model to improve its accuracy by removing columns and measuring the accuracy of the models
  • Decided to go with Logistic Regression as my chosen algorithm as it had the highest accuracy
  • Tested the model to predict the customer status on th etesting set
  • Built a simple website using Flask

Skills/Languages Used: Python & other ML and data cleaning libraries like numpy, pandas, scikit learn
